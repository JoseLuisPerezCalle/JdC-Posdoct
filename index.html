This research focuses on integrating advanced machine learning techniques with spectroscopic methods to tackle challenges in food safety and fire investigation. By utilizing tools such as FTIR, Vis-NIR, and E-nose systems, it enhances the detection of adulteration in products like honey and juices, where traditional methods often fall short. Additionally, it improves the analysis of ignitable liquid residues (ILRs) in fire debris by automating and optimizing data interpretation. This approach not only increases reliability and efficiency but also promotes more sustainable and accessible practices.

The main purpose is to demonstrate the potential of combining machine learning with analytical techniques to transform complex data into actionable insights, benefiting industries, research, and quality control processes.

These studies integrate advanced spectroscopic techniques such as FTIR, Vis-NIR, and IMS with machine learning algorithms to rapidly and accurately detect and quantify adulterations in juices. The graphs, including spectra, Principal Component Analysis (PCA), and Hierarchical Cluster Analysis (HCA), offer a clear and accessible presentation of the key findings.

Principal Components Analysis (PCA)

PCA was applied to reduce the dimensionality of the spectroscopic data, allowing for easier visualization of the variability between the juice samples. As shown in Figure 2, PCA successfully separates grape juice from the other juices, particularly in FTIR (Figure 2A) and IMS (Figure 2B), where the grape juice occupies a distinct position along the first principal component (PC1). This highlights significant spectral differences between grape juice and the other juices, such as apple, pineapple, and orange.

In contrast, Vis-NIR (Figure 2C) displays a different trend, with grape juice clustering closer to apple juice in the principal component space, suggesting greater similarity in their spectral profiles. This may indicate that Vis-NIR has a lower capacity to distinguish between grape juice and the other juices compared to FTIR and IMS. Since grape juice was used as the adulterant, this observation may have implications for the detection of adulteration.

Overall, all three techniques demonstrated varying degrees of success in differentiating the juices based on their principal components. FTIR and IMS showed clearer separations, while Vis-NIR displayed some overlap between certain juice types.

Hierarchical Cluster Analysis (HCA)

The HCA results, presented in Figure 3, show dendrograms for each of the three techniques, revealing patterns consistent with the PCA findings. Both FTIR (Figure 3A) and IMS (Figure 3B) successfully group the juice samples into distinct clusters. FTIR, in particular, perfectly separates the four main juice types into individual clusters, with further subclusters identifying samples by brand. IMS also shows a similar trend, though the grape juice is further divided into two distinct clusters based on the brand, indicating potential differences in their spectral profiles.

Vis-NIR (Figure 3C), however, shows a more diffuse classification pattern. While some grouping according to juice type is observed, the separation is not as clear as in FTIR and IMS. Similar to the PCA results, apple juice appears closer to the grape juice adulterant, and some subclusters are mixed. Pineapple juice is mostly grouped together, although one sample, "P_CA_1_R1", is separated from the main cluster.

Applications

In this section we present the different web applications performed for the authentication of food (juices and honey) as well as for the identification of ILRs in fire debris, by different spectroscopic techniques. These applications have been previously published in the different articles that make up this Doctoral Thesis (See results page).

It should be noted that all applications have a similar operation. Through an easy-to-use interface, users can rapidly and objectively characterize samples without the need for their own database or prior knowledge of ML algorithms. The operation is simple: users upload an Excel file generated from the spectroscopic analysis of the sample, and by clicking the "submit" button, predictions are made immediately. Furthermore, a downloadable example file (by clicking the "Download" button) is included to help users become familiar with the platform's functionality.Explore our interactive Shiny applications below:

Application 1: FT-IR-Based Juice Adulteration Detection Using Machine Learning

This application is a prototype designed to detect and quantify adulteration in fruit juices using FTIR combined with ML algorithms, specifically SVR and SVM models, that achieved the best performance metrics. Users can upload their FTIR data in Excel or CSV format and receive predictions on the adulteration levels. The system is designed to improve as more samples are processed, enhancing its accuracy in identifying adulteration across different juice types.​

Application 2: Fire reports: Data from Junta de Leon y Castilla

This application has been created to facilitate the visualization of wildfire reports provided by the Junta de Castilla y León. It features an interactive map where users can view information about different wildfire reports, as well as select the month and year of the incidents. This tool enhances the accessibility and understanding of wildfire data, offering a more intuitive way to explore trends and patterns over time. The information used in this application is sourced from the following link: Junta de Castilla y León Wildfire Data.https://analisis.datosabiertos.jcyl.es/explore/dataset/.​

Application 3: Rapid Detection and Quantification of Juice Adulteration Using Machine Learning and NIR Spectroscopy data

This application is designed to detect and quantify adulteration in fruit juices using Vis-NIR data combined with ML models, specifically RF for adulteration detection and SVR for adulteration quantification. The user can upload their data in Excel or CSV format, and the system will process the FT-IR spectra to predict adulteration levels. As the application processes more data, its accuracy improves by learning from the variability in the samples. This tool provides a fast, automated solution for the detection of juice-to-juice adulterations, enhancing quality control in the beverage industry.​

Application 4: Detection of Ignitable Liquids Residues Using HS-GC-IMS and Machine Learning

This application provides insights into the detection of ILRs in fire debris using a combination of HS-GC-IMS and ML algorithms. The technique offers an innovative and automated alternative to traditional methods like GC-MS, allowing for the effective discrimination between ILRs even in the presence of interfering substrates. By applying machine learning tools such as RF and SVM, the models achieve an accuracy of up to 100% for detecting and classifying ILRs, making it a powerful tool in forensic fire investigations​.

Application 5: Portable Electronic Nose System for Fire Debris Analysis

This web application enables users to characterize fire debris samples by detecting and identifying ignitable liquids (ILRs) through machine learning algorithms applied to data from a portable eNose system. Users can upload sample data in an Excel format, and the application provides rapid and automated predictions, offering a user-friendly alternative to traditional.

Application 6: Automated Quantification of Juice Adulteration Using HS-GC-IMS and Machine Learning

This application provides an advanced methodology for the detection and quantification of adulteration in fruit juices, specifically focusing on the addition of cheaper grape juice to other juices like orange, pineapple, and apple. The technique integrates HS-GC-IMS with ML algorithms, including SVR, PLS, and RF. The Boruta algorithm is employed to optimize variable selection, improving model performance. The best results were achieved using the SVR model, with an R² of 0.987 and an RMSE of 1.831, demonstrating high accuracy in quantifying adulteration levels. An interactive web application has been developed to allow users to upload their data and automate the detection process (using the best performance models), making it accessible for routine quality control in the food industry.​

Application 7: Honey Adulteration Detection Using Vis-NIR Spectroscopy and Machine Learning

This application is designed to detect and quantify adulteration in honey using Vis-NIR data in combination with ML algorithms. Honey samples from two botanical origins (orange blossom and sunflower) are tested, and adulteration with low-cost honey is introduced at various percentages (5% to 50%). By employing SVM model, the application achieves 100% accuracy in detecting adulteration. For quantification, SVR is used, as it demonstrates excellent performance, with an R² of 0.991 and an RMSE of 1.894, offering a powerful tool for automated quality control in the honey industry.​
